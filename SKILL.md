# 文档归类整理工具

将work文件夹中的文档按规则分类整理到raw文件夹，支持增量同步和客户映射表管理。

## 使用方式

```bash
# 默认增量模式
python scripts/organize.py

# 强制全量刷新
python scripts/organize.py --full

# 仅更新客户映射表
python scripts/organize.py --mapping-only
```

## 源→目标映射

| 源路径 | 目标路径 |
|--------|----------|
| work/产品手册/* | raw/产品功能/ |
| work/产品方案/* | raw/通用业务方案/ |
| work/客户知识库/{客户}/蓝图/* | raw/优秀客户方案/{客户}/蓝图/ |
| work/客户知识库/{客户}/蓝图/* | raw/客户档案/{客户}/蓝图方案/ |

## 客户映射

- 映射表保存在 `raw/_mapping.json`
- 基于客户主数据（商务信息档案/客户主数据）生成
- 运维工单匹配时，只归类在映射表中的客户
- 不在映射表中的客户工单忽略

## 增量同步

- 状态记录在 `raw/_sync_state.json`
- 记录每个文件的hash和时间戳
- 只处理新增或修改的文件

## 数据完整性

处理Excel时：
- 读取所有sheet，不遗漏
- 保留原始列名和数据
- 拆分时按客户分组，每个客户一个文件
- 合并时汇总所有数据，不丢失任何行
